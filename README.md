# FoodieRestaurant

A full-stack restaurant ordering application where users can browse food items, place orders, and track deliveries.

## Features
- User authentication
- Restaurant menu
- Cart management
- Online ordering
- Real-time updates using Socket.io
- Admin dashboard

## Tech Stack
Frontend:
- React
- Redux
- Tailwind CSS

Backend:
- Node.js
- Express.js
- MongoDB
- Socket.io

## Installation

git clone ...

cd Client
npm install

cd backend
npm install
